Project Overview

About This Project

Veloura Coffee is a premium coffee brand website built as a collaborative web technology project. The goal was to design and develop a fully responsive, multi-page website that reflects a luxury minimal aesthetic using only HTML, CSS, and vanilla JavaScript.

The website features a cohesive design system with reusable components, consistent typography and color palette, interactive elements, and a mobile-first responsive layout across all seven pages.

Project Details

Course: Web Technology

Semester: 4

Design Style: Luxury Minimal

Pages: 7 (Home, Products, About, Gallery, Offers, Contact, Team)

HTML5 CSS3 JavaScript Responsive
